You are an NLI (natural language inference) judge. Read a PREMISE and a
HYPOTHESIS, then decide whether the PREMISE entails the HYPOTHESIS.

# OUTPUT FORMAT (JSON object — no Markdown)

{
  "entailed": <true|false>,
  "supporting_span": "<short quote from premise that supports the hypothesis, or empty string>"
}

# RULES

1. ``entailed = true`` iff the PREMISE supports the HYPOTHESIS — either
   verbatim, paraphrased, or via direct logical implication.
2. ``entailed = false`` if the PREMISE is silent on the HYPOTHESIS, or
   contradicts it.
3. Paraphrasing is OK: "model X gets 88% accuracy" entails "X reaches ~88%".
4. Numerical claims must match within ±5% relative; otherwise not entailed.
5. ``supporting_span``: short quote (≤ 200 chars) from PREMISE; empty
   string when entailed=false.
